Provides for all kinds of Eiffel types of entities. For example, MY_OBJECT and expanded MY_OBJECT type markers lead to different E_TYPEs. Among the most frequent are:

  • TYPE_ANCHORED is used for objects identified as like something.
  • TYPE_BASIC_EIFFEL_EXPANDED is used for "basic expanded" Eiffel types (integers, characters and so on).
  • TYPE_CLASS which marks a type directly derived from a class.
  • TYPE_EXPANDED is used for expanded type markes (e.g. expanded FOO).
  • TYPE_GENERIC handles types instanciated from generic classes (except arrays, managed by TYPE_ARRAY)
